# Sports Hall Booking System using Service Portal-ServiceNow

This project focuses on developing a comprehensive Sports Hall Booking System within the ServiceNow platform, leveraging the Service Portal as the primary user interface. The system will allow users to view available time slots for various sports halls, submit booking requests, and receive automated notifications regarding the status of their requests. Administrators will be able to manage the sports hall inventory, approve or reject booking requests, and generate reports on booking trends and resource utilization. The project will involve designing a custom data model, configuring service catalog items, implementing workflow automation, and customizing the Service Portal to provide a seamless user experience.
